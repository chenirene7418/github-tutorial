## Git vs. GitHub

**_What is GIT?_**  
Git is a software where you can keep track of your changes.  
Git keeps "screenshots" of your code or version control. 

**_What is Github?_**  
Github is a cloud or internet.   
Github is where you can easily collabrate on the same file.
